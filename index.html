Fayvor.io

Python, PHP, Javascript, HTML, CSS, AWS(EC2, CodeCommit, RDS), mySQL

We created the basis for the startup Fayvor. We turned their pencil and paper business into an online one by hosting their server and services on an EC2 server using AWS. All of the code was stored on CodeCommit with the database being set up using RDS. Then we created python scripts that would run on their website that would take the information inside google sheets linked to surveys. Each survey response would be added to our decision tree mathcing algorithm to determine the best possible match between a caretaker and a client to ensure the best service.
